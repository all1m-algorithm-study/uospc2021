# 전구 끄기 - EASY

## 문제

$1$번부터 $N$번까지 전구가 원형으로 나열되어 있다. 즉 아래 그림과 같이, $i(1 \leq i \leq N-1)$번 전구는 $i+1$번 전구와 서로 인접하며, $N$번 전구는 $1$번 전구와 서로 인접하다. 초기 상태에서 어떤 전구는 켜져있으며 어떤 전구는 꺼져있다.

<div style="text-align:center;"><img style="width:50%;" src="./images/bulb-intro.png" /></div>

당신은 임의의 한 전구에서 출발해 다른 전구까지 시계방향 또는 반시계 방향으로 한 칸씩 이동하면서, **시작 전구부터 끝 전구까지를 포함하는 구간 내의 모든 전구를 끄려고 한다.** 그런데 먼 거리를 이동하는 것은 힘든 일이므로, 최대한 짧은 거리를 이동하면서 $N$개의 전구를 모두 끄려고 한다. 이때 **모든 전구를 끄기 위해 필요한 최소 이동 거리**를 구하는 프로그램을 작성하시오.

여기서 이동거리는 시작점에서 출발해 끝점에 도착하기까지 인접한 전구로 이동한 횟수와 같다. 예를 들어 아래 그림처럼 이동하면서 전구를 모두 끄면 이동거리는 $4$이고, $4$보다 적은 거리를 이동하면서 모든 전구를 끄는 것은 불가능하다. **(한 전구만 끄는 경우에는 이동거리가 $0$임에 유의하라.)**

<div style="text-align:center;"><img style="width:75%;" src="./images/bulb-easy.png" /></div>

## 입력

첫째 줄에는 전구의 개수 $N$이 주어진다. $(1 \leq N \leq 100,000)$
둘째 줄에는 $N$개 전구의 상태가 주어진다. 둘째 줄의 $i$번째 값 $c_i$는 각 전구의 점등 여부를 나타낸다. $c_i$가 $1$이라면 해당 전구가 켜져있다는 것을 의미하며, $c_i$가 $0$이라면 해당 전구가 꺼져있다는 것을 의미한다.

## 출력

첫째 줄에 전구를 모두 끄기 위한 이동 거리의 최솟값을 한 줄에 출력한다.

## 예제 입력 1

```
7
1 1 0 1 0 0 1
```

## 예제 출력 1

```
4
```

지문에서 설명한 상황이다.

## 예제 입력 2

```
3
1 0 0
```

## 예제 출력 2

```
0
```

$1$번 전구에서 출발하여 $1$번 전구에서 끝나는 상황이므로 이동 거리는 0이다.
